MONOLITH

A brutalist poster annual for fictional civic events. Six posters. One grid. Disciplined excess.

01

The Brief

Monolith — a brutalist poster compilation. An annual of oversized typographic posters for fictional civic events.

Signature: disciplined excess — massive Archivo Black headlines cut by a hard grid, scroll-snap poster pages, hover reveals the grid skeleton. Chaos gridded underneath.

Palette: raw concrete #b9b5ad, ink #1c1a17, warning orange #e8551a. Type: Archivo Black + Inter Tight. Prove: loud can be rigorous.

Upgrade ideas mentioned in the brief: print-registration marks; a poster that rebuilds itself letter by letter; concrete texture via CSS gradients only.

02

Signature Technique

Three techniques define this build: scroll-snapped full-viewport poster pages, a CSS-only concrete texture applied to flat color backgrounds, and a hover-revealed 12-column grid overlay that makes the skeleton visible.

The concrete texture is the most important detail — it distinguishes these posters from flat-color brutalism by adding the material imperfection of actual cast concrete, using only layered repeating-linear-gradient declarations at three angles. No images, no canvas.

Concrete texture — CSS gradients only (style.css)

/* Three diagonal passes at different angles and periods */
.poster--renewal {
  background-color: var(--concrete);
  background-image:
    repeating-linear-gradient(
      -55deg,
      transparent 0, transparent 4px,
      rgba(28,26,23,0.018) 4px, rgba(28,26,23,0.018) 5px
    ),
    repeating-linear-gradient(
      35deg,
      transparent 0, transparent 7px,
      rgba(255,255,255,0.01) 7px, rgba(255,255,255,0.01) 8px
    ),
    repeating-linear-gradient(
      82deg,
      transparent 0, transparent 11px,
      rgba(28,26,23,0.01) 11px, rgba(28,26,23,0.01) 12px
    ),
    linear-gradient(168deg, #c0bcb4 0%, var(--concrete) 45%, #b2aea7 100%);
}

Hover grid skeleton — opacity transition (style.css)

.grid-overlay {
  position: absolute;
  inset: 0;
  opacity: 0;
  background-image:
    repeating-linear-gradient(
      to right,
      var(--grid-col) 0, var(--grid-col) 1px,
      transparent 1px, transparent calc(100% / 12)
    ),
    repeating-linear-gradient(
      to bottom,
      var(--grid-row) 0, var(--grid-row) 1px,
      transparent 1px, transparent calc(100% / 8)
    );
  transition: opacity 0.45s cubic-bezier(0.4, 0, 0.2, 1);
}

.poster:hover .grid-overlay,
.poster:focus-within .grid-overlay {
  opacity: 1;
}

Letter scramble — MONOLITH cover title (main.js)

/* Each letter cycles through random uppercase chars
   before resolving — staggered 85 ms per character */
function frame(now) {
  letters.forEach((span, i) => {
    const tStart  = startTime + i * STAGGER;  // 85 ms stagger
    const tSettle = tStart + SETTLE;          // 420 ms scramble window

    if (now < tStart)  return;            // not started
    if (now < tSettle) {
      span.textContent = CHARS[Math.floor(Math.random() * CHARS.length)];
      return;
    }
    span.textContent = span.dataset.ch;    // resolved
  });
  if (anyPending && !document.hidden) requestAnimationFrame(frame);
}

04

Critique History

01

First pass — structure & grid

  • Established scroll-snap full-viewport poster pages; confirmed zero console errors at all poster scroll positions.
  • Cover title too large — escaped viewport width at narrow breakpoints; clamped with clamp(4.5rem, 15vw, 18rem) and added mobile override.
  • Noise poster lacked the waveform element entirely; added deterministic bar heights via JS to fill the dead zone between body copy and penalty list.
  • Transit poster: right-panel content had no bottom anchor — added justify-content: space-between to distribute tag, title, route, stats, date, and authority across full panel height.
  • Grid overlay missing on dark-background posters (Ink posters used orange grid lines that were invisible on concrete posters); added per-poster --grid-col and --grid-row custom properties.
Complexity upgrade: Print registration marks in all four corners — each a real crosshair with a circle punch, drawn purely in CSS using pseudo-elements and radial-gradient. They change colour per poster (dark on light, light on dark).
02

Second pass — type & contrast

  • Urban Renewal poster: "RENEWAL" word shift (padding-left) was colliding with the fine-print marginalia column at 1280 px — reduced marginalia width to clamp(150px, 18%, 220px) and added pointer-events: none.
  • River District poster on mobile: orange text on ink background and concrete text on ink background both passed contrast, but the orange block behind the title was showing through on small screens making "DISTRICT" unreadable. Fixed by switching the entire mobile poster to a full-orange background (background-color: var(--orange)) and inverting all text to ink.
  • Works poster: list items had inconsistent vertical rhythm — switched from gap to justify-content: space-between on .works__list so items always fill the full right column height regardless of viewport.
  • Side nav link contrast on orange (Transit) poster: added text-shadow: 0 1px 5px rgba(0,0,0,0.7) so the concrete-toned numbers read against the orange field.
Complexity upgrade: Live-status pulsing dot on all "Active" work items — a 5 px circle appended via ::after, animated with a cubic-bezier scale+opacity loop. Respects prefers-reduced-motion (dot stays fully visible, static).
03

Third pass — polish & mobile

  • Transit mobile: the giant "7" numeral was overflowing its absolute container on 390 px — switched to a static top anchor with transform: translateX(-50%) and moved the content panel to margin-top: auto with a semi-transparent orange background strip.
  • Cover mobile: padding-right retained the desktop nav-clear extra padding, leaving the right margin unbalanced — removed the extra 3.5rem on mobile.
  • Works mobile: left "PUBLIC" wordmark column was empty at 390 px after column collapse — constrained to height: clamp(5rem, 15vh, 12rem) and set overflow: hidden so the massive type clips cleanly at the divider.
  • Noise penalty values on mobile were wrapping; fixed with flex-wrap: wrap and reduced gap.
  • Extra-small breakpoint (≤ 390 px) added: --edge: 1rem, --edge-v: 1.25rem.
Complexity upgrade: Keyboard navigation between posters via Arrow and Page keys — JS intercepts ArrowDown/PageDown/ArrowUp/PageUp and calls scrollIntoView({ behavior: 'smooth' }), degrading to 'auto' in reduced-motion contexts.
