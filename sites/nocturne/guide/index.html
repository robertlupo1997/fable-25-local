Nocturne

Field Guide to Moths of the Invented World

The Brief

What was asked for

18. nocturne — a field guide to moths. Dark botanical plates for invented species. Signature: procedurally drawn moths — bilateral symmetry from seeded parameters (wing shape, eyespots, dust); they steer toward a draggable lamp with real steering behaviors; plates render as engraved SVG. Palette: night #131521, page #e9e6db, lamp #ffd47e, wing dust #9aa4c7. Type: Faustina + Nunito Sans. Prove: generative creatures with taxonomy-plate elegance.

BRIEFS.md · Wave 3 · Fable 25 Local

The core tension in the brief is taxonomic precision versus living motion — engraved plates carry the stillness of a museum drawer; the canvas scene places the same creatures in flight. The guide resolves this by treating the two registers separately: SVG plates use ink-stroke aesthetics and calibrated scale bars; the canvas simulation uses steering behaviors and wing-flap dynamics.

Signature Technique

Procedural SVG moths with bilateral symmetry

Each of the six species is defined by a parameter object — wing dimensions, tip angle, eyespot positions, dust density, scallop flag — and a seeded PRNG. The SVG is generated at runtime: every wing outline, cross-hatch pattern, eyespot ring, dust stipple, and scale bar is built from these parameters. No pre-drawn artwork is used.

Bilateral symmetry is enforced by a side argument (±1) passed to every drawing function. The same bezier control points that produce the right forewing, negated, produce the left:

main.js — foreWingPath() · bilateral symmetry via side = ±1

function foreWingPath(fw, fh, tipAngle, side) {
  const s = side; // +1 = right, -1 = left
  const cx = BODY_CX;
  const cy = BODY_TY;
  const tipY = cy - tipAngle * fh * 0.44;

  return [
    `M ${cx} ${cy - 4}`,
    `C ${fmtN(cx + s*fw*0.30)} ${fmtN(cy - fh*0.70)},`,
    `  ${fmtN(cx + s*fw*0.62)} ${fmtN(tipY - fh*0.14)},`,
    `  ${fmtN(cx + s*fw)} ${fmtN(tipY)}`,
    `C ${fmtN(cx + s*fw*1.03)} ${fmtN(tipY + fh*0.36)},`,
    `  ${fmtN(cx + s*fw*0.87)} ${fmtN(cy + fh*0.60)},`,
    `  ${fmtN(cx + s*fw*0.52)} ${fmtN(cy + fh*0.80)}`,
    // Return to body centre
    `C ${fmtN(cx + s*fw*0.25)} ${fmtN(cy + fh*0.90)},`,
    `  ${fmtN(cx + s*5)} ${fmtN(cy + fh*0.65)},`,
    `  ${cx} ${fmtN(cy + fh*0.35)}`,
    `Z`
  ].join(' ');
}

The same symmetry argument passes through every sub-function: eyespotSVG, venationLines, dustSVG, antennaPath. Each function multiplies its x-offsets by s, so specifying one wing fully specifies the other. Dust and venation use separate PRNG instances per side to avoid mirror-identical scatter.

i

Cross-hatching: SVG <pattern> at 45° and 135°, clipped to wing outlines via <clipPath>. Opacity differs between hatch angles to create directional shading near the body.

ii

Eyespots (ocelli): three concentric rings per ocellus — outer stroke, pale fill, dark pupil — positioned by fractional u/v coordinates within the wing bounding box, so wing size does not affect eyespot proportion.

iii

Scale bar: calibrated to actual specimen wingspan. Nocturia grandipennis at 85–112 mm renders a longer bar than Selenus pulverulentus at 28–36 mm, using the formula scaleBarPx = fw × 20 / wsMid.

iv

Bipectinate antennae: a curved SVG path with computed normals at each interval used to orient perpendicular barbs — the feathered appearance of moth (vs butterfly) antennae.

The canvas simulation assigns each moth a canvasScale derived from its real wingspan (Nocturia at 1.45×, Selenus at 0.60×), so larger species are visibly larger in flight. Steering uses a wander circle on the current heading, blended with an arrival force toward the lamp and a tangential orbit force that prevents pile-up within 80 px.

main.js — spawnMoths() · moon-phase-weighted species pool (pass 2 upgrade)

// Determine current moon phase bucket (0–7)
const moonP = getMoonPhase() / 29.53058867;
let moonBucket;
if      (moonP < 0.04 || moonP > 0.96) moonBucket = 0; // New Moon
else if (moonP < 0.22)                moonBucket = 1; // Waxing Crescent
else if (moonP < 0.28)                moonBucket = 2; // First Quarter
// …(full moon = 4, waning crescent = 7)

// Build weighted pool — each species appears (weight) times
const pool = [];
SPECIES.forEach(sp => {
  const w = (sp.phaseWeights || [])[moonBucket] ?? 2;
  for (let j = 0; j < w; j++) pool.push(sp);
});

Critique Pass History

What each pass found and changed

I

Stage 1 · Initial build + first critique

  • Canvas moths rendered as undifferentiated blobs — wing bezier redesigned to distinct forewing + hindwing pair with separate stroke and alpha
  • Mobile moon text overlapped subtitle — .moon-note and .scene-subtitle hidden on viewports below 600 px
  • Plate SVG had a 240 px max-width constraint leaving dead space at card edges — removed, SVG now fills card width
  • Moths clumped at lamp origin — orbit tangential force increased; outward repel force added inside 45 px radius
  • <em> used for site title — corrected to <h1>; section headings downgraded to <h2>
Complexity upgrade

Calibrated scale bar added to every specimen plate SVG — 10 mm calculated from actual species wingspan midpoint using scaleBarPx = fw × 20 / wsMid. Smallest species (Selenus, 28–36 mm) renders a noticeably shorter bar than the largest (Nocturia, 85–112 mm).

II

Stage 2 · Color identity + phase ecology

  • All canvas moths read as near-identical gray-blue — impossible to distinguish species in flight, defeating the field-guide premise
  • Moths near the lamp still crowded center despite orbit force — acceptable but noted; orbit radius increase would push them further out
Complexity upgrade

Moon-phase species filtering: each species now carries a phaseWeights[8] array mapping to the eight moon phase buckets. Lethia pallida (dawn species, "lamp is rarely helpful") peaks at waning crescent; Selenus pulverulentus peaks at new and waxing crescent; Veluma ocellata (strongly lamp-attracted) peaks at full moon. The canvas scene reflects tonight's actual phase. Canvas colors were simultaneously made dramatically distinct per species: iridescent slate-blue for Veluma, pale straw-cream for Lethia, dark charcoal for the cryptic Umbria tenebrosa, warm amber for the large Nocturia grandipennis.

III

Stage 2 · Species identification on hover

  • Size differentiation per species: canvas moth size now scales by canvasScale (Nocturia at 1.45×, Selenus at 0.60×), making wingspan differences visible in flight
  • Zero console errors and zero page errors across desktop and mobile at all scroll positions
Complexity upgrade

Hovering near any canvas moth fades in a two-line identification label — common name in small caps above, binomial italic in dust-blue below — using a custom-eased alpha ramp. The label smoothly fades out when the cursor moves away. Adds the discovery layer a real field guide implies: the scene is not ornament, it is the index.
