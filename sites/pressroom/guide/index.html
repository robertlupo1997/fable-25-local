§ 2

Signature Technique

The central proof: a pixel-level letterpress simulation in vanilla canvas that earns its texture through computation, not stock photos. Every print is unique — different grain seed, slight ink variation per stroke pixel, colour-offset misregistration when loose-registration mode is on.

  1. Paper base. Canvas is filled with oat (#ede5d6) and overlaid with per-pixel fractal grain noise (±9 per channel), seeded fresh each print.
  2. Text measurement. Binary-search fits the Rozha One string to 74% canvas width, preventing overflow on long composing-stick lines.
  3. Offscreen render. Text is drawn to a hidden canvas in the target ink colour (or rubine offset + union blue for loose registration).
  4. Ink squash composite. Pixel loop: if alpha < 0.70, boost by 1.28× — simulates ink bleeding into paper at antialiased stroke edges. Random ±16 noise per ink pixel adds variation within thick strokes.
  5. Impression shadow. A second offscreen pass at 0.07 alpha, multiply blend, +1.5 px offset — the deboss shadow of real impression depth.
  6. Crop marks. Four 12 px corner marks at 0.25 alpha give the sheet a press-sheet register.
  7. Loose registration. Rubine drawn first at +3.5/+2.5 px offset, union blue at 86% opacity on top — two-colour misprint, charming and intentional.

main.js · renderLetterpress() — the pixel composite loop

// 4. Pixel-level letterpress composite (ink squash + variation)
var offData  = oCtx.getImageData(0, 0, W, H);
var od = offData.data;
var mainData = ctx.getImageData(0, 0, W, H);
var md = mainData.data;

for (var i = 0; i < total; i++) {
  var alpha = od[i * 4 + 3] / 255;
  if (alpha < 0.015) continue;

  // Ink squash: boost alpha at antialiased edges
  // simulates ink bleeding into paper fibres
  var ea = alpha < 0.70
    ? Math.min(1.0, alpha * 1.28)
    : alpha;

  var noise = rnd(16); // ±16 per ink pixel
  var idx = i * 4;
  md[idx]   = Math.round(lerp(md[idx],   clamp(oR + noise * 0.55), ea));
  md[idx+1] = Math.round(lerp(md[idx+1], clamp(oG + noise * 0.55), ea));
  md[idx+2] = Math.round(lerp(md[idx+2], clamp(oB + noise * 0.55), ea));
}
