A lab demonstration site for an impossible material. Physics simulation as the hero; light theme done with discipline.

The brief

Build a demonstration site for EMG-905 ferrofluid, a fictional lab specimen. Signature technique: 2D ferrofluid simulation — metaballs via marching-squares on canvas, spiking into Rosensweig peaks under a draggable magnet; field lines drawn faintly beneath. Palette: lab white #f2f4f6, ferro black #0a0a0c, coil copper #c96f2e. Type: Chivo + Chivo Mono. Prove: physics simulation as the hero, light theme done with discipline.

Signature technique: metaball pool + Bezier spike comb

The ferrofluid simulation runs two parallel systems. First, 18 circular blobs with spring physics model the bulk fluid mass — they cluster under gravity with weak magnetic attraction from the draggable magnet. Second, an off-screen pixel-marching loop samples the scalar field of all blob contributions and composites only the fluid region onto the main canvas using per-pixel alpha: 

// Offscreen pixel loop — alpha-composite over grid + field lines
const v = sampleField((col + 0.5) * scX, (row + 0.5) * scY);
const t = smoothstep(THRESHOLD_LO, THRESHOLD_HI, v);
d[i]     = 10;  // ferro black (R)
d[i + 1] = 10;  // ferro black (G)
d[i + 2] = 12;  // ferro black (B)
d[i + 3] = Math.round(t * 255); // alpha = fluid density

The t = 0 region above the pool stays fully transparent — the grid and field lines underneath remain visible. The Rosensweig spikes are a separate Bezier-comb canvas path drawn below the metaball layer; their tips extend into the transparent region and show through, while their bases merge seamlessly into the opaque fluid body.

Spike geometry

Spikes are a single closed canvas path — not individual shapes. A Gaussian envelope computes the height of each peak based on its horizontal distance from the magnet centerline, then Bezier curves carve smooth valleys between peaks: 

function spikeH(i) {
  const off = (i - (numS - 1) / 2) * space;
  return h * Math.exp(-0.5 * (off / (space * numS * 0.29)) ** 2);
}

Field-line integrator

Twelve field lines integrate a dipole field numerically (4 px per step), starting from evenly spaced angles around the magnet. Each step follows the local field vector — one component from a real magnetic pole at the magnet position, one from a mirror image above (simulating the return path). The result is the characteristic closed-loop dipole pattern visible behind the fluid body.

Upgrade: peak-index annotations

After drawing the spike path, a second pass labels each prominent Rosensweig peak with a copper specimen number (01–09) at its tip — small tick marks and monospace type in the same lab-document voice as the copy below the fold. The labels only appear when spike height exceeds 28% of the maximum, suppressing noise on the Gaussian tails.
