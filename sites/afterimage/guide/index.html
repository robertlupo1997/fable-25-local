SECTION 01

The Brief

Brief #24 of FABLE 25 LOCAL. The brief was received verbatim; no brief was added or modified.

24. `afterimage` — a photocopy zine on seeing. Five spreads on perception, illustrated by a live WebGL image-processing rig applied to procedurally generated "photographs." Signature: post-processing as art direction — ordered dithering, CMYK halftone separation, chromatic aberration, video-feedback tunnels, each spread one effect with its recipe. Palette: zine white #f5f5f2, ink #111111, process magenta #ec008c, cyan #00aeef, yellow #ffe600. Type: Space Grotesk + Space Mono. Prove: shader post-processing taught like a zine, punk but precise. Upgrade ideas: a copier "degrade" button (each press re-photocopies the page); registration-drift on scroll; staple shadows.
#F5F5F2 #111111 #00AEEF #EC008C #FFE600

The brief's core tension: technical shader education delivered as a physical zine. Every spread had to work as both a legible procedural "photograph" and a readable recipe for the effect. The aesthetic mandate — "punk but precise" — meant embracing visible artifacts (strong chromatic fringing, dithered portraits) rather than softening them.

SECTION 02

Signature Technique: Ping-Pong FBO Video Feedback

The most structurally complex piece is Spread 04 — the video feedback tunnel. Real video feedback (pointing a camera at its own monitor) requires a true memory between frames. A standard single-pass WebGL2 renderer reads from the canvas it writes to — a hazard. The solution is a ping-pong FBO pair.

Two RGBA8 textures (A and B) and two framebuffer objects live in GPU memory. Each frame, the feedback shader reads from the source texture and writes to the destination FBO — never the one it reads. Then a blit shader copies the destination to the visible canvas. The source and destination swap every frame.

SEED PASS — runs once at init, writes concentric rings + spokes into texA:

// FRAG_SEED — seeds the initial pattern into texA
void main(){
  vec2 p = v_uv - 0.5;
  float r = length(p);
  float rings  = 0.5 + 0.5 * sin(r * 32.0);
  float spokes = 0.5 + 0.5 * cos(atan(p.y, p.x) * 4.0);
  float core   = smoothstep(0.1, 0.0, r);
  float v = rings * 0.22 * smoothstep(0.5, 0.0, r)
          + spokes * 0.12 * smoothstep(0.45, 0.18, r) + core;
  fragColor = vec4(v * 0.25, v * 0.08, v, 1.0);
}

FEEDBACK PASS — runs every frame, reads srcTex → writes dstFBO:

// FRAG_FEED — zoom × rotate × decay × inject new signal
void main(){
  vec2 p = v_uv - 0.5;

  // Shrink UV by 1/zoom so sampled image appears larger next frame
  float zoom = 1.0 / 1.019;
  float ang  = 0.005; // radians per frame
  float cs = cos(ang), sn = sin(ang);
  vec2 rp = vec2(cs*p.x - sn*p.y, sn*p.x + cs*p.y) * zoom;

  vec4 prev  = texture(u_prev, clamp(rp + 0.5, 0.0, 1.0));
  vec3 faded = prev.rgb * 0.962; // decay factor

  // Slow hue-drift tint keeps the tunnel colour from bleaching
  float th  = u_t * 0.25;
  vec3 tint = vec3(0.97 + sin(th)*0.03,
                   0.94 + cos(th*0.7)*0.04,
                   0.985 + sin(th*0.5)*0.015);

  // New signal: rotating spokes + central pulse at low amplitude
  float r2  = length(p);
  float spk = (0.5 + 0.5*sin(atan(p.y,p.x)*3.0 + u_t*1.4))
              * smoothstep(0.48, 0.28, r2) * smoothstep(0.08, 0.2, r2) * 0.06;
  float pulse = smoothstep(0.05, 0.0, r2) * (0.5 + 0.5*sin(u_t*2.6)) * 0.1;

  vec3 sig = vec3(spk*0.35, spk*0.1, spk)
           + vec3(pulse*0.9, pulse*0.65, 0.0);

  fragColor = vec4(clamp(faded * tint + sig, 0.0, 1.0), 1.0);
}

SWAP LOGIC — JavaScript orchestration (from main.js):

// Each frame: read from srcTex, write to dstFBO, blit, swap
const srcTex = current === 0 ? texA : texB;
const dstFBO = current === 0 ? fboB : fboA;
const dstTex = current === 0 ? texB : texA;

// Feedback pass: srcTex → dstFBO
gl.bindFramebuffer(gl.FRAMEBUFFER, dstFBO);
gl.useProgram(progFeed);
gl.bindTexture(gl.TEXTURE_2D, srcTex);
gl.drawArrays(gl.TRIANGLE_STRIP, 0, 4);

// Blit to canvas: dstTex → null framebuffer
gl.bindFramebuffer(gl.FRAMEBUFFER, null);
gl.useProgram(progBlit);
gl.bindTexture(gl.TEXTURE_2D, dstTex);
gl.drawArrays(gl.TRIANGLE_STRIP, 0, 4);

current = 1 - current; // swap

The result after ~120 frames: a psychedelic purple-magenta tunnel that develops like a long-exposure photograph. The IntersectionObserver gates each renderer so the feedback loop only runs when its canvas is visible — preventing the tunnel from "building" off-screen.
