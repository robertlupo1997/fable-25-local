Commission Brief

What was asked for

"a museum of impossible objects. Five artifacts that cannot exist, raymarched. Signature: SDF raymarching in fragment shaders — a Klein bottle reliquary, a Penrose triangle in bronze, a shadow that casts an object; museum-label typography treats each absurdity with total scholarly sobriety. Palette: charcoal #17161a, label ivory #efe9dd, brass #ad8f52, violet #8a63d2. Type: Cardo + Questrial. Prove: raymarching as a gallery medium; copywriting with a straight face." — FABLE 25 Briefs, Wave 3, No. 14

Upgrade ideas listed in the brief: acquisition numbers and provenance; a rotating plinth interaction; a "do not touch" easter egg that ripples the SDF. All three were implemented across the build and iteration passes.

Signature Technique

SDF raymarching in GLSL ES 1.0

Every artifact is rendered entirely inside a WebGL fragment shader — no meshes, no geometry buffers. A ray is cast per pixel; the signed distance field (SDF) for each impossible object tells the marcher how far it can step before it must check again. When distance drops below 0.0015 world units, we have a hit. The surface normal is recovered by numerical central differences on the same SDF — four extra evaluations, no explicit normals stored.

Five independent shaders share a common header (TOP) and a marcher template (MARCHER) injected at compile time. The shared march() loop references whatever map*() function is defined in each shader:

main.js — shared marcher (GLSL) Ref. technique · MARCHER template 
float march(vec3 ro, vec3 rd) {
  float t = 0.1;
  for (int i = 0; i < 88; i++) {
    float d = mapKlein(ro + rd * t);   // swap per shader
    if (d < 0.0015) return t;
    if (t > 20.0) break;
    t += d * 0.85;                      // underrelax: safer at thin features
  }
  return -1.0;                          // miss
}

vec3 getNormal(vec3 p) {
  float e = 0.003;
  return normalize(vec3(
    mapKlein(p + vec3(e,0,0)) - mapKlein(p - vec3(e,0,0)),
    mapKlein(p + vec3(0,e,0)) - mapKlein(p - vec3(0,e,0)),
    mapKlein(p + vec3(0,0,e)) - mapKlein(p - vec3(0,0,e))
  ));
}

The Klein bottle SDF is built from six primitive capsules and two tori, blended with a smooth-minimum (smin) operator. The topology is described rather than rigorously derived — the neck rises through the torus hole, arcs over the rim in two segments, and re-enters the body from the exterior, suggesting the impossible self-intersection:

main.js — Artifact I SDF (GLSL) Ref. No. 1923-VII-004 · FRAG_1 
float mapKlein(vec3 p) {
  float t = uTime * 0.18;
  p.xz *= rot2(t);
  p.y -= 0.15;

  // Torus body — the outer "bottle" surface
  float body  = sdTorus(p + vec3(0,0.38,0), vec2(1.02, 0.40));
  float rim   = sdTorus(p - vec3(0,0.06,0), vec2(0.54, 0.13));

  // Neck rises through the torus hole (center axis)
  float neck  = sdCapsule(p, vec3(0,-1.45,0), vec3(0, 1.0, 0), 0.11);

  // Over-top arc — curves right, then back over the rim
  float arc1  = sdCapsule(p, vec3(0,   1.00, 0), vec3(0.55, 0.70, 0), 0.11);
  float arc2  = sdCapsule(p, vec3(0.55,0.70, 0), vec3(1.10, 0.10, 0), 0.10);

  // Entry — tube re-enters the torus body from outside
  // This is the impossible crossing: interior becomes exterior
  float entry = sdCapsule(p, vec3(1.10, 0.10, 0), vec3(0.88,-0.38, 0), 0.10);

  float d = body;
  d = smin(d, rim,   0.16);   // smooth blend — no hard seam
  d = smin(d, neck,  0.10);
  d = smin(d, arc1,  0.08);
  d = smin(d, arc2,  0.08);
  d = smin(d, entry, 0.08);

  // Pass 3 upgrade: click ripple (uClick: 0 = just clicked, 1 = gone)
  d += sin(length(p) * 12.0 - uClick * 18.0)
     * 0.018 * clamp(1.0 - uClick, 0.0, 1.0);

  return d;
}

The uMouse uniform (Pass 2) shifts the camera look-at target by up to ±0.25 units horizontally and ±0.15 vertically as the cursor moves across each canvas, creating a parallax orbit. The uClick uniform (Pass 3) drives a spherically-expanding ripple through the distance field — amplitude 0.018 × frequency 12 keeps the Lipschitz bound below 0.22, safe for 88-step marching without visible banding.

Critique Register

What each pass found, what changed

Pass I Build + first critique
  • Klein topology unclear: the neck arced over the torus but the re-entry point was ambiguous. Fixed: two explicit arc capsules (arc1, arc2) added to show the impossible crossing, plus an entry capsule meeting the torus equator from outside.
  • Canvas cramped: aspect ratio was 57.5% (canvas appeared squeezed). Fixed: raised to 60% (5 : 3).
  • Stairway loop unconvincing: 12 steps were insufficient for the Penrose staircase illusion at this camera angle. Fixed: 16 steps, giving one full 360° rise closing to step 1.
  • Upgrade implemented: "Do not touch" easter egg — clicking any canvas flashes the sign brass-coloured for 1.2 s. upgrade
Pass II Polish + parallax
  • Artifact numerals invisible: .artifact-numeral used color: var(--rule) (#3a3740 on #17161a) — essentially black on black. Fixed: changed to rgba(173,143,82,0.38), a dim brass that reads as decorative but legible.
  • Canvas display case flat: the wrapper border gave no depth impression. Fixed: added box-shadow: inset 0 1px 0 rgba(239,233,221,0.07), 0 8px 32px rgba(0,0,0,0.45) — a faint top-edge glass highlight and a base drop shadow.
  • Canvas cursor gave no affordance: nothing signalled interactivity. Fixed: cursor: crosshair on all canvases.
  • Upgrade — mouse parallax: uniform vec2 uMouse added to all five shaders. Hovering any canvas shifts the camera look-at target by the smoothed cursor position (lerp factor 0.07/frame). Mobile scroll left untouched — tracking is hover-only on non-touch devices. upgrade
Pass III SDF ripple + guide
  • Penrose Bronze renders correctly: bronze material with GGX specular, the three impossible right-angle joints visible. No changes to the shader — the visual reads as intended.
  • "Do not touch" only flashed a sign: the easter egg was cosmetic, not physical. Fixed: uniform float uClick passes seconds since last click (clamped 0–1) into every map function. A spherical ripple sin(length(p)*12.0 - uClick*18.0) * 0.018 * clamp(1.0-uClick, 0.0, 1.0) deforms the SDF for ~0.8 s after each click — the geometry itself responds to touch. Shadow No. 7 uses a floor-plane variant (length(p.xz)*8.0, amplitude 0.04) so the shadowless ground undulates. upgrade
  • Guide page: this document, written in the collection's visual language — same palette, same typefaces, honest findings only.

Acknowledgements & Limitations

What the museum cannot guarantee

The Klein bottle SDF is a description, not a derivation. A true Klein bottle has no mathematical closed-form SDF in Euclidean 3-space — by definition. The visual language of the impossible crossing is suggested by primitive composition; topology is implied rather than proven.

The Penrose triangle holds the depth illusion at a specific camera angle. Rotation reveals the physical seam where the arms disconnect. This is, in the committee's view, not a defect.

The Faceted Tesseract performs 32 capsule and 16 sphere SDF evaluations per fragment per marching step. On older mobile GPUs with step count reduced to 56, the surface may show stepping artefacts at high-curvature regions. The museum accepts no liability for conclusions reached about the fourth dimension.
