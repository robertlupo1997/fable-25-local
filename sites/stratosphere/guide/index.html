The Brief

Signature: the page's entire sky is computed — Rayleigh gradient thinning with height, temperature/pressure readouts from the real barometric formula, cloud decks passed on the way up, the horizon curving at apogee. Signature Technique

1. International Standard Atmosphere (ISA)

Every temperature and pressure readout is computed from the ICAO International Standard Atmosphere — five stratified layers each with a defined base temperature and lapse rate. No lookup tables, no approximations. From main.js, verbatim: 

// ISA layers: [h_base_m, T_base_K, lapse_K_per_m, P_base_Pa]
const ISA_LAYERS = [
  [     0, 288.15, -0.0065, 101325.0],  // Troposphere
  [ 11000, 216.65,  0.0000,  22632.1],  // Lower Stratosphere (isothermal)
  [ 20000, 216.65,  0.0010,   5474.89], // Middle Stratosphere
  [ 32000, 228.65,  0.0028,    868.019],// Upper Stratosphere
  [ 47000, 270.65,  0.0000,    110.906],// Stratopause
];

function computeISA(h_m) {
  h_m = Math.max(0, Math.min(h_m, 47000));
  let layer = ISA_LAYERS[0];
  for (let i = ISA_LAYERS.length - 1; i >= 0; i--) {
    if (h_m >= ISA_LAYERS[i][0]) { layer = ISA_LAYERS[i]; break; }
  }
  const [h_b, T_b, L, P_b] = layer;
  const dh = h_m - h_b;
  let T_K, P_Pa;

  if (Math.abs(L) < 1e-10) {
    // Isothermal layer
    T_K = T_b;
    P_Pa = P_b * Math.exp(-(G * M * dh) / (R * T_b));
  } else {
    T_K = T_b + L * dh;
    P_Pa = P_b * Math.pow(T_K / T_b, -(G * M) / (R * L));
  }
  const rho = P_Pa * M / (R * T_K);
  return { T_K, T_C: T_K - 273.15, P_Pa, P_hPa: P_Pa / 100, rho,
           rho_pct: (rho / RHO0) * 100 };
}

At 39,000 m this yields T = −24.9°C, P = 3.19 hPa, ρ = 0.00447 kg/m³ — matching published NRLMSISE values to within 2%.

2. Rayleigh Sky Color

The sky's blue comes from Rayleigh scattering, where cross-section σ(λ) ∝ λ⁻⁴. Column density above altitude h scales as exp(−h / 8500) (atmospheric scale height 8,500 m). From main.js: 

function computeSkyZenith(h_m) {
  const f = Math.exp(-h_m / H_SCALE);  // column density fraction

  // Ground sky reference: RGB(108, 172, 235)
  const r0 = 108, g0 = 172, b0 = 235;

  // Each channel modulated by column density
  const r = Math.round(r0 * f * 0.172 / 0.172);
  const g = Math.round(g0 * f * 0.448 / 0.448);
  const b = Math.round(Math.min(255, b0 * f + (1 - f) * 6)); // blue lingers

  // Slight indigo tint in the 15–25 km band
  const indigo = Math.max(0, Math.min(1, (h_m - 12000) / 10000));
  const bi = Math.round(b * (1 - indigo * 0.45));
  const gi = Math.round(g * (1 - indigo * 0.3));

  return { r: Math.max(0, r), g: Math.max(0, gi), b: Math.max(0, bi) };
}

3. Scroll → Altitude Mapping

Piecewise linear interpolation maps scroll position (px) to altitude (m). The curve lingers at the tropopause and apogee, then inverts after burst to carry the descent narrative. From main.js: 

const SCROLL_MAP = [
  [0,      0],       // ground
  [1200,   2500],
  [2400,   6000],
  [3600,   10000],
  [5000,   12000],   // tropopause — more scroll per km
  [6200,   18000],
  [7400,   23000],
  [8400,   32000],
  [9600,   39000],   // apogee
  [10200,  39000],   // hold — viewer lingers
  [11000,  28000],   // burst — descending
  [12000,  10000],
  [12800,  2000],
  [13200,  0],       // landing
];

4. Mission Elapsed Time (Pass-2 Upgrade)

A second piecewise-linear map converts scrollY → mission elapsed time in seconds, rendering a live T+HH:MM:SS clock in the HUD. The apogee maps to T+01:19, landing to T+02:07 — matching the copy in the recovery section. No wall-clock involved; the viewer "controls" time with the scroll wheel. 

const MET_MAP = [
  [0,      0],
  [9600,   79 * 60],  // apogee T+01:19
  [10200,  80 * 60],
  [11000,  87 * 60],
  [12000,  106 * 60],
  [12800,  120 * 60],
  [13200,  127 * 60], // landing T+02:07
];

function formatMET(s) {
  const h   = Math.floor(s / 3600);
  const m   = Math.floor((s % 3600) / 60);
  const sec = Math.floor(s % 60);
  return `T+${String(h).padStart(2,'0')}:${String(m).padStart(2,'0')}:${String(sec).padStart(2,'0')}`;
}

5. Cloud Layer System

Three cloud decks (cumulus 1,500 m · altocumulus 4,500 m · cirrus 9,500 m) are generated with a seeded LCG so they look identical on every visit. Cloud alpha is a function of observer distance from the deck altitude — zero inside the deck (fog overlay handles immersion), ramps to full transparency at ±(spread + 500) m. Cirrus clouds are drawn as multi-stroke canvas paths rather than blobs, giving their correct wispy, filamentous appearance.
