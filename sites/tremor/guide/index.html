Tremor

A century of earthquakes — builder's notes

The brief

Build a scroll-driven data essay on 100 years of major seismic events using real USGS catalog data. Signature aesthetic: seismograph — needle-drawn canvas traces, each decade a chapter, magnitude rendered as physical amplitude you feel in the layout. Palette: seismogram paper #faf6ee, ink #26221c, event red #c8102e, depth blue #274b8f. Type: Spectral + Martian Mono. Prove: honest data storytelling with real cited data.

Data

All 413 events are M7.5+ earthquakes from the USGS FDSN Event Service, fetched on 2026-07-08 and baked into data.js as a static const QUAKES = [...]. No runtime network call. The query was: minmagnitude=7.5, starttime=1926-01-01, endtime=2026-01-01. Each event stores only: mag, place, time, depth, lon, lat — 47 KB total.

Pre-1960 coverage is instrumentally sparse; this is acknowledged in a completeness note on affected decade sections, and discussed in the intro text. The honest acknowledgment is the editorial stance, not a workaround.

Signature technique — the seismograph trace

Every trace is computed from scratch on a <canvas> using a damped sine wave per event, overlaid into a single continuous Float32Array of y-offsets. The amplitude formula uses a 1.5-power law to preserve the physical intuition that magnitude is logarithmic, while keeping M9.5 spikes visually dramatic without crowding out M7.5 events:

// Amplitude: (mag − 7.0)^1.5 × scale
// M7.5 → ~11 px  M8.0 → ~30 px  M9.0 → ~85 px  M9.5 → ~119 px
function magAmp(mag, scale) {
  return Math.pow(Math.max(mag - 7.0, 0.01), 1.5) * scale;
}

// Per-event spike: damped oscillation (seismometer physics)
const decayPx = 0.038;   // e-fold decay per pixel
const freq    = 0.24;    // radians per pixel

for (let dx = 0; dx <= spikeLen; dx++) {
  const env = Math.exp(-decayPx * dx);
  const osc = Math.sin(freq * dx + Math.PI * 0.5);
  trace[ix] -= amp * env * osc;  // upward deflection
}

The amplitude scale is computed per-decade so the biggest event in each decade always uses ~85% of the canvas height above the baseline. This means a quiet 1980s trace looks appropriately modest, while the 1960s section — M9.5 + M9.2 — towers over everything else. The M9.5 Valdivia spike clips the canvas top intentionally: the needle went off the chart.

Other design decisions

Scroll velocity → needle jitter. The scroll event feeds a velocity estimate into a decay loop; any currently-visible decade trace redraws with a subtle high-frequency perturbation applied to the y-values. The effect is small — max ±8 px at high velocity — but makes the page feel alive when scrolling fast. Gated off in reduced-motion mode.

IntersectionObserver animation. Traces draw left-to-right with a custom quadratic ease when each decade section enters the viewport. Duration is 1600 ms plus 4 ms per event, so dense decades draw slightly slower. The needle-tip glows red while in flight.

Depth cross-section. A 54 px canvas below each trace plots events by focal depth (0–700 km). Three background bands distinguish shallow / intermediate / deep; dot size scales with magnitude; opacity deepens with depth. One event in the 1970s Peru section reaches 644 km — it appears as the sole dark dot near the 700 km line.

Reduced motion. All canvas traces draw statically at full progress; no IntersectionObserver timing, no jitter, no needle-tip glow. The page is a dignified static composition, not a blank or broken one.
